## 🚀 在Mini Studio中使用ER15-1400

### 方法1: 创建新项目时选择
1. 点击"新建项目"
2. 在"机械臂类型"下拉框中选择"ER15-1400"
3. 完成项目创建，系统自动加载ER15-1400参数

### 方法2: 在现有项目中切换
1. 进入"机械臂"配置页面
2. 点击"选择预设模型"下拉框
3. 选择"ER15-1400"
4. 系统自动更新DH参数和关节限制

### 方法3: 手动配置参数
1. 进入"机械臂"配置页面
2. 手动输入上述DH参数和关节限制
3. 系统会自动识别并显示ER15-1400专用3D模型

## 🔧 技术实现细节

### URDF解析过程
1. 读取ER15-1400的URDF文件
2. 提取关节变换矩阵
3. 转换为标准DH参数表示
4. 关节限制从弧度转换为度数

### 3D模型渲染
- 基于Three.js的WebGL渲染
- 实时关节角度更新
- 碰撞检测边界框
- 材质和光照优化

### 运动学计算
- 正运动学: DH参数 → 末端位姿
- 逆运动学: 目标位姿 → 关节角度
- 雅可比矩阵: 速度映射关系
- 奇异点检测: 工作空间边界
