# LoongEnv - 工业机械臂智慧控制生态

## 项目结构

```
LoongEnv/
├── docs/                    # 项目文档
│   ├── PRD.md              # 产品需求文档
│   ├── architecture.md     # 架构设计文档
│   └── api/                # API文档
├── packages/               # 核心模块
│   ├── studio/            # LoongEnv-Studio (设计平台)
│   ├── twin/              # LoongEnv-Twin (数字孪生)
│   ├── net/               # LoongEnv-Net (智能控制)
│   ├── box/               # LoongEnv-Box (实时控制)
│   └── eye/               # LoongEnv-Eye (监控系统)
├── shared/                # 共享组件
│   ├── types/             # 类型定义
│   ├── utils/             # 工具函数
│   └── config/            # 配置文件
├── tools/                 # 开发工具
│   ├── build/             # 构建脚本
│   └── deploy/            # 部署脚本
└── tests/                 # 测试文件
    ├── unit/              # 单元测试
    ├── integration/       # 集成测试
    └── e2e/               # 端到端测试
```

## 快速开始

1. 查看产品需求文档: `docs/PRD.md`
2. 了解架构设计: `docs/architecture.md`
3. 开始开发: 选择对应的模块目录

## 开发规范

- 遵循语义化版本控制
- 使用约定式提交信息
- 代码覆盖率 >80%
- 所有PR需要代码审查
