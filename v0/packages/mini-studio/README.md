# Mini LoongEnv-Studio 快速启动指南

## 🚀 快速开始

### 1. 环境要求
- Node.js >= 18.0.0
- npm >= 8.0.0

### 2. 安装和运行
```bash
# 进入Mini Studio目录
cd packages/mini-studio

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

访问 http://localhost:3000 即可使用Mini LoongEnv-Studio

## 📋 功能概览

### 核心工作流
1. **项目管理** - 创建和管理机械臂项目
2. **机械臂配置** - 设置DH参数和关节限制
3. **轨迹设计** - 3D空间中设计运动轨迹
4. **仿真预览** - 实时预览机械臂运动

### 主要特性
- ✅ 纯前端实现，无需后端服务
- ✅ 数据自动保存到本地存储
- ✅ 3D可视化和交互操作
- ✅ 实时仿真和动画播放
- ✅ 配置导入导出功能
- ✅ 响应式设计，适配不同屏幕

## 🎯 使用教程

### 第一步：创建项目
1. 点击"新建项目"
2. 输入项目名称和描述
3. 选择机械臂类型（当前支持6轴工业机械臂）

### 第二步：配置机械臂
1. 切换到"机械臂"标签
2. 配置DH参数（a, d, θ, α）
3. 设置关节限制（最小/最大角度）
4. 查看3D机械臂模型

### 第三步：设计轨迹
1. 切换到"轨迹"标签
2. 在3D空间中点击添加轨迹点
3. 编辑轨迹点的位置和名称
4. 查看轨迹路径可视化

### 第四步：仿真预览
1. 切换到"仿真"标签
2. 点击"开始"按钮启动仿真
3. 观看机械臂运动动画
4. 查看性能指标和统计数据

### 第五步：导出配置
1. 点击右上角"导出配置"按钮
2. 下载JSON格式的配置文件
3. 可用于实际机械臂系统部署

## 🎮 操作指南

### 3D场景操作
- **鼠标左键拖拽** - 旋转视角
- **鼠标右键拖拽** - 平移视图
- **鼠标滚轮** - 缩放视图
- **点击空白区域** - 添加轨迹点
- **点击轨迹点** - 选择轨迹点
- **拖拽轨迹点** - 修改位置

### 键盘快捷键
- **R** - 重置视角
- **G** - 显示/隐藏网格
- **Delete** - 删除选中轨迹点
- **Ctrl+Z** - 撤销操作（计划中）
- **Ctrl+S** - 保存项目（计划中）
