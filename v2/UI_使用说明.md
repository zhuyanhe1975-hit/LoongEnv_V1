# 机器人控制界面使用说明

## 🎯 回答您的问题

### 1. 机器人3D模型显示位置

机器人的3D模型现在显示在以下位置：

- **仪表板页面** (主页) - 右上角区域
- **监控页面** - 左侧主要区域

3D模型功能：
- 实时显示机器人关节角度
- 支持鼠标旋转、缩放、平移操作
- 显示关节角度数值覆盖层
- 连接状态指示

### 2. 虚拟机器人连接方法

有多种方式连接虚拟机器人：

#### 方法1：通过顶部导航栏
1. 点击顶部导航栏右侧的 **"未连接"** 按钮
2. 在弹出的连接管理对话框中点击 **"连接虚拟机器人"**
3. 等待1秒钟连接完成

#### 方法2：通过快速操作面板
1. 在仪表板页面找到 **"快速操作"** 卡片
2. 点击 **"连接机器人"** 按钮
3. 在弹出的对话框中配置并连接

#### 方法3：通过设置菜单
1. 点击顶部导航栏的设置图标
2. 选择连接管理选项

### 3. 虚拟机器人特性

✅ **完全虚拟化** - 无需真实硬件
- 模拟ER15-1400六轴机械臂
- 实时关节运动仿真
- 简化的正向运动学计算
- 可配置的传感器噪声

✅ **实时数据流** - 50Hz更新频率
- 关节位置、速度、力矩
- 末端执行器位姿
- 系统负载监控
- 安全状态检测

## 🚀 快速开始

### 第一次使用
1. 确保开发服务器运行：`npm run dev`
2. 访问 http://localhost:3001/
3. 点击 **"连接机器人"** 按钮
4. 观察3D模型开始运动

### 界面导航
- **仪表板** - 系统概览和3D模型
- **监控** - 详细的系统监控和3D可视化
- **规划** - 轨迹规划工具
- **调优** - 参数调优界面
- **设置** - 系统配置

## 🎮 虚拟机器人控制

### 连接配置
在连接管理对话框中可以配置：
- **更新频率** (1-100Hz) - 控制数据更新速度
- **仿真速度倍率** (0.1-10x) - 控制运动速度
- **传感器噪声** - 开启/关闭噪声模拟

### 自动运动模式
连接后，虚拟机器人会自动执行：
- 周期性关节运动
- 正弦波轨迹跟踪
- 随机扰动模拟

### 状态监控
实时显示：
- 6个关节的角度、速度、力矩
- 末端执行器位姿 (x,y,z,rx,ry,rz)
- CPU、内存、温度等系统指标
- 安全状态和错误代码

## 📝 使用提示

1. **首次连接**：点击任何 "连接" 按钮即可
2. **观察运动**：连接后3D模型会自动开始运动
3. **调整视角**：拖拽鼠标旋转，滚轮缩放
4. **监控数据**：查看关节角度数值和系统状态

## 🔍 故障排除

### 连接失败
- 检查浏览器控制台错误
- 刷新页面重试
- 确保JavaScript已启用

### 3D模型不显示
- 检查WebGL支持
- 更新浏览器版本
- 检查硬件加速设置

### 数据不更新
- 检查连接状态
- 重新连接虚拟机器人
- 查看浏览器网络面板
